# Proyecto_vd_2

Proyecto de video juego desarrollado bajo el motor de Unity.

Version demo del proyecto.

Jugabilidad:
Golpe: Enter
Golpe con daga: Enter
Agacharse: Ctrl - derecho
Salto: Espacio
Caminar - Correr: W
Caminar derecha: D
Caminar Izquierda: A
Caminar atras: S
Elimar daga: Q
Guardad Juego: G
Cargar Juego: C
